# 2026-03-27 チャット UI 実装スタック方針

## 日付

- 2026-03-27

## テーマ

- チャット UI と OpenAI 接続をどの構成で実装するか

## 結論

- チャット UI は `assistant-ui` を採用する
- 現在の実装は `assistant-ui + AI SDK OpenAI provider` で接続する
- 将来的に必要があれば、サーバー側を公式 `openai` SDK へ寄せる余地は残す

## 理由

- `assistant-ui` は Next.js 上でチャット画面を素早く立ち上げやすい
- 現時点では最短で動くチャット UI を優先したい
- まず会話体験とタイムライン連携の土台を作ることが重要

## 影響範囲

- `/api/chat` で OpenAI 接続を受ける
- 環境変数として `OPENAI_API_KEY` と `OPENAI_MODEL` を使う
- 左カラムのチャットはダミー表示ではなく実際に API を叩く構成になる

## 保留事項

- サーバー側を公式 `openai` SDK へ完全移行するか
- 会話内容から予定候補を構造化してタイムラインへ反映する方法
