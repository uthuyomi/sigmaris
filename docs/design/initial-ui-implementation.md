# 初期 UI 実装メモ

## 実装日

- 2026-03-27

## 実装内容

- `Next.js 16 + TypeScript + Tailwind CSS` のアプリ基盤を作成
- ルート画面にチャット中心レイアウトを実装
- 24 時間の縦タイムライン UI を実装
- 時間粒度切り替えボタン `5 / 10 / 15 / 30 / 60 分` を実装
- モバイル用の下部ナビゲーションを仮実装

## 画面構成

- 左カラム: アシスタントとの予定調整チャット
- 右カラム: 24 時間タイムラインと予定ブロック
- 補助パネル: 選択中の予定の簡易編集表示

## 現時点の性質

- まだデモ UI であり、Supabase や認証との接続は未実装
- タイムラインの粒度切り替えは表示用の状態管理まで実装済み
- 予定データはダミーデータで表示している

## 次に必要な実装

- Supabase Auth とログイン画面の接続
- Supabase から予定データを取得してタイムラインへ反映
- 予定の新規作成、移動、リサイズの操作実装
- チャット入力と予定提案ロジックの接続
