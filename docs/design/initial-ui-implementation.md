# 初期 UI 実装メモ

## 実装日

- 2026-03-27

## 実装内容

- `Next.js 16 + TypeScript + Tailwind CSS` のアプリ基盤を作成
- 画面を `ホーム / チャット / タイムライン / カレンダー / 設定` に分割
- 24 時間の縦タイムライン UI を独立画面として実装
- 月表示カレンダー UI を追加
- 時間粒度切り替えボタン `5 / 10 / 15 / 30 / 60 分` を実装
- 共通ナビゲーションを実装

## 画面構成

- ホーム: 各ビューへの入口
- チャット: アシスタントとの予定調整
- カレンダー: 月単位の俯瞰表示と日付選択
- タイムライン: カレンダーで選んだ日の24時間詳細調整
- 設定: 認証やデフォルト設定の置き場

## 現時点の性質

- まだデモ UI であり、Supabase や認証との接続は未実装
- タイムラインの粒度切り替えは表示用の状態管理まで実装済み
- 予定データは共通モックデータで表示している
- カレンダーからタイムラインへ日付指定付きで遷移できる

## 次に必要な実装

- Supabase Auth とログイン画面の接続
- Supabase から予定データを取得してタイムラインへ反映
- タイムラインから前後日の移動導線
- 予定の新規作成、移動、リサイズの操作実装
- チャット入力と予定提案ロジックの接続
