# 画面構成メモ

## 実装日

- 2026-03-27

## ページ構成

- `/`: ホーム。各ビューへの入口と状態要約
- `/chat`: チャット専用画面
- `/timeline`: カレンダーで選んだ日付の24時間タイムライン詳細
- `/calendar`: 月表示カレンダー
- `/settings`: 認証やデフォルト設定の置き場

## 実装方針

- 共通の外枠は `AppShell` にまとめる
- モック予定データは共通化して各ビューで再利用する
- モバイルでも同じナビゲーションで主要画面へ移動できるようにする
- 日付選択はカレンダー、時刻調整はタイムラインに役割分担する

## 次にやること

- モックデータを Supabase データへ置き換える
- タイムラインから前後の日付へ移動する導線を入れる
- チャット結果をタイムラインとカレンダーへ反映する
