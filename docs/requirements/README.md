# 要件ドキュメント

このカテゴリでは、機能要件、非機能要件、ユースケース、仕様メモを管理します。

## 置く内容

- 何を作るか
- 誰が使うか
- どんな条件で動くべきか
- どこまでを今回の対象にするか

## ファイル作成の例

- `feature-shift-creation.md`
- `user-role-definition.md`
- `non-functional-requirements.md`
